Class Documentation¶

class Command¶

The Command class is the mechanism to pass all commands to the unit (including Read and Write Register) and track the unit’s respective response.

Subclassed by VN::AsyncOutputEnable, VN::DebugSwitches, VN::FirmwareUpdate, VN::KnownAccelerationDisturbance, VN::KnownMagneticDisturbance, VN::Microcontroller, VN::PollBinaryOutputMessage, VN::Reset, VN::RestoreFactorySettings, VN::ServicePortSwitch, VN::SetBootLoader, VN::SetFilterBias, VN::SetInitialHeading, VN::WriteSettings
